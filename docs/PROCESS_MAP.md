# 发电全流程与粒子表达

## 事实分层

- **厂级公开事实**：浙江公开报告指向乐清市磐石镇的浙江浙能温州发电有限公司；2014 年验收资料记载 5、6 号燃煤机组和“低氮燃烧 + SCR（尿素）”脱硝系统。后续公开资料显示厂内还有 7、8 号机组。所有年份、容量和改造状态在展示前必须再次核对。
- **行业通用流程**：煤粉锅炉、汽轮发电、烟气脱硝/除尘/脱硫、灰渣分流和水处理来自生态环境部门公开技术资料。
- **实习观察**：只进入私有底稿；公开案例使用泛化名称、相对值或合成数据。

## 六条流程

### 1. 燃料与空气

煤接卸/储存 → 输煤 → 碎煤 → 原煤仓 → 磨煤机 → 煤粉燃烧器 → 炉膛。

送风机 → 空气预热器 → 二次风/燃尽风 → 炉膛；一次风同时承担煤粉干燥与输送。实际厂区的接卸方式、设备数量和路径需由公开资料或现场许可信息确认。

### 2. 汽水循环

凝汽器 → 凝结水泵 → 低压加热器 → 除氧器 → 给水泵 → 高压加热器 → 省煤器 → 水冷壁/蒸发系统 → 过热器 → 高压缸 → 再热器 → 中低压缸 → 凝汽器。

不同机组可能采用汽包炉或直流炉，模型必须用 `unitVariant` 区分，不能把所有机组画成同一结构。

### 3. 机械能与电能

蒸汽推动汽轮机 → 同轴发电机产生电能 → 主变压器升压 → 配电装置/升压站 → 电网。部分电能经厂用变压器返回泵、风机、磨煤机等辅机。

### 4. 烟气治理

炉膛 → 过热/再热受热面 → 省煤器 → SCR 脱硝 → 空气预热器 → 高效除尘 → 湿法脱硫 → 烟囱。

公开验收资料确认温电 5、6 号机组采用低氮燃烧与 SCR，尿素热解后产生含氨混合气，在催化剂作用下将 NOx 转化为氮气和水。具体机组的设备顺序与超低排放改造状态要按最新公开文件更新。

### 5. 灰渣、石膏与废水

炉底渣 → 脱水/渣仓 → 综合利用；除尘飞灰 → 灰库 → 综合利用；湿法脱硫副产石膏 → 建材利用。工业废水按来源分类处理、回用或达标处置，不在模型中简化为“直接排放”。

### 6. 仪控与工作闭环

现场传感器/变送器 → I/O → DCS 趋势与报警 → 人员判断 → 工作票/隔离 → 检修或更换 → 复测 → 消缺关闭 → 经验沉淀。

这条“信息流”是本作品区别于普通发电动画的关键，它把实习中的热电偶、压力表、热电阻、液位雷达和轴承测温串进产品故事。

## 粒子编码

| 流程 | 建议颜色/形态 | 动画含义 | 禁止误导 |
|---|---|---|---|
| 煤/煤粉 | 深灰颗粒，煤粉更细 | 方向与相对输送速度 | 不用粒子数表示真实吨位 |
| 空气 | 青绿色轻粒子 | 风向、一次风/二次风分支 | 不伪造真实风量 |
| 蒸汽 | 白色高亮粒子 | 主汽、再热汽与做功后的状态变化 | 不用颜色冒充精确温压 |
| 水 | 蓝色连续粒子 | 凝结水、给水、冷却水回路 | 先区分闭式/开式系统 |
| 烟气 | 灰色粒子，治理后透明度降低 | 沿烟道流动与污染物被移除 | “变干净”不等于零排放 |
| 电力 | 黄色脉冲 | 发电、升压、送出和厂用电回路 | 不表示真实电压相位 |
| 灰渣/石膏 | 棕灰色分支颗粒 | 固体副产物去向 | 综合利用比例需有来源 |
| 控制信号 | 品红色短脉冲 | 采集、报警、指令与闭环 | 不复刻内部控制逻辑 |

## 数据模型原则

每个节点必须包含 `id`、`name`、`system`、`inputs`、`outputs`、`evidenceLevel` 和 `sourceIds`。每条边必须包含介质、方向、可视化样式和验证状态。场景坐标属于表现层，不能反向被当成真实厂区坐标。
