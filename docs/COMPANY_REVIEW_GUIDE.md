# 实习成果阅读指南

## 领导与企业读者：三分钟路径

1. 阅读首页定位、四项成果和16条脱敏实习事件。
2. 打开 `#product`，查看FieldTrace如何从实际日志形成问题定义、MVP和指标。
3. 打开 `#lab`，查看一个流程节点和一个脱敏异常复盘。
4. 查看“观察现场、追踪链路、校验证据、转成产品”的学习方法与公开边界。

重点观察：

- 在陌生工业场景中的学习速度与主动性；
- 跨流程、仪表和信息链路的系统理解；
- 面对不完整信息时的证据意识；
- 从真实痛点定义用户问题、MVP边界和验证指标的产品思维；
- 将零散见闻整理成可展示成果的表达和交付能力。

## 亲友：一分钟路径

1. 先看首页成果概览和16条实际日志。
2. 进入流程实验室，跟随粒子流理解发电流程。
3. 回到首页阅读“这段成长”，了解我在实习中的具体变化。

本项目展示学习成果，不披露生产系统、敏感设备参数或未经授权的现场材料。
