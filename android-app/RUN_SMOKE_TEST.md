# Android 作品集冒烟测试

使用虚构、脱敏测试内容。不要导入真实机组号、参数、现场截图、人员信息或内部资料。

1. 安装 `app/build/outputs/apk/debug/app-debug.apk`，确认“我的、现场、数据、轨迹、更新”五个页面均可打开。
2. 在任意普通页面左右滑动，确认页面和几何导航同步切换；纵向滚动不应误触换页。
3. 在“我的”修改称呼，重启后确认仍然保留；进入沉浸模式后确认顶栏、底栏和系统栏隐藏，按返回键可退出。
4. 在“数据”拖动和双指缩放三维响应曲线，点选数据点并切换95%、90%数据，确认读数和二维切片同步变化。
5. 在“轨迹”旋转、缩放和点选节点，确认下方现场事件随选择更新。
6. 关联一个虚构视频，确认可重新打开；断开网络后重新启动，内置内容仍可访问。
7. 在“更新”导入 [portfolio-content.example.json](content-pack/portfolio-content.example.json) 的修改副本，再恢复内置内容，确认两次切换均不会删除称呼和视频关联。

通过标准：冷启动无崩溃、横向滑屏与纵向滚动不冲突、三维交互持续流畅、内容包可恢复、离线可读、本地数据保持。
